autor: Daniel Justa
verze: 1.0

- jedná se o jednoduchou textovou hru
- tématem hry je středověk a fantasy
- hráč si vybere jednu ze 3 postav a jeho cílem je zabít draka, který žije v podzemí
- hráč vyhraje po úspěšném zabití draka
- ve hře je systém nakupování elixírů (elixirSily, elixirZivotu) za zlaťáky
- elixirSily přidá permanentně sílu a elixirZivotu zase životy
